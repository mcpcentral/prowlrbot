# Support

## Getting Help

| Need | Where to Go |
|------|-------------|
| **Questions about setup or usage** | [GitHub Discussions (Help)](https://github.com/prowlrbot/prowlrbot/discussions/categories/help) |
| **Bug reports** | [GitHub Issues](https://github.com/prowlrbot/prowlrbot/issues/new?template=bug_report.yml) |
| **Feature requests** | [GitHub Issues](https://github.com/prowlrbot/prowlrbot/issues/new?template=feature_request.yml) |
| **Security vulnerabilities** | See [SECURITY.md](https://github.com/prowlrbot/prowlrbot/blob/main/SECURITY.md) |
| **Documentation** | [prowlrbot.com/docs](https://prowlrbot.com/docs) |

## Before Opening an Issue

1. Search [existing issues](https://github.com/prowlrbot/prowlrbot/issues) to avoid duplicates
2. Check the [documentation](https://prowlrbot.com/docs)
3. Try the latest version: `pip install --upgrade prowlrbot`
4. Include your Python version, OS, and relevant config (redact secrets)
