ROAR Protocol

One protocol. Any agent.

Discover agents, fetch cards, send tasks — from your browser. Point at any ROAR endpoint.

ROAR (Reliable Open Agent Relay) is a 5-layer agent communication protocol: identity (DIDs), discovery, transport, message exchange, and streaming. ProwlrBot and compatible agents speak it over HTTP and WebSocket. pip install roar-protocol
