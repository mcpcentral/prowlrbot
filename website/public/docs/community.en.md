# Bug Reports & Community

Report bugs, request features, or connect with the ProwlrBot community.

---

## Report a Bug

Found something broken? We want to know.

1. Check [existing issues](https://github.com/ProwlrBot/prowlrbot/issues) to avoid duplicates
2. Open a [new issue](https://github.com/ProwlrBot/prowlrbot/issues/new) with:
   - What you expected to happen
   - What actually happened
   - Steps to reproduce
   - Your environment (OS, Python version, ProwlrBot version)

## Security Vulnerabilities

For security issues, please see our [Security Policy](https://github.com/ProwlrBot/prowlrbot/blob/main/SECURITY.md). We acknowledge reports within 48 hours and patch critical issues within 7 days.

## Feature Requests

Have an idea? Open a [GitHub Discussion](https://github.com/ProwlrBot/prowlrbot/discussions) in the Ideas category. If the community supports it, we'll convert it to a tracked issue.
